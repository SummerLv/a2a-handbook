# A2A Protocol Checklist

> 部署前检查清单

## 🔐 安全检查

### Agent Card 安全

- [ ] 不包含敏感信息（内部 URL、Token 名称等）
- [ ] 描述已清理，无 Prompt Injection 风险
- [ ] URL 使用 HTTPS
- [ ] 版本号正确更新
- [ ] 认证要求已声明

### 认证与授权

- [ ] 所有端点需要认证（公开 Agent Card 除外）
- [ ] Token 有过期时间
- [ ] Scope 权限检查在后端实现
- [ ] 无硬编码凭证
- [ ] 敏感操作需要二次验证

### 输入验证

- [ ] 所有输入经过验证和清理
- [ ] 文件大小限制
- [ ] MIME 类型检查
- [ ] 文件内容扫描
- [ ] URL 白名单验证

---

## 📊 性能检查

### 资源限制

- [ ] 请求大小限制（如 10MB）
- [ ] 连接超时设置（如 30s）
- [ ] 读取超时设置（如 60s）
- [ ] 并发连接限制
- [ ] 速率限制配置

### 缓存优化

- [ ] Agent Card 缓存策略
- [ ] HTTP 缓存头正确设置
- [ ] 连接池配置

---

## 📝 日志与监控

### 日志记录

- [ ] 请求日志（方法、参数、响应）
- [ ] 错误日志（异常堆栈）
- [ ] 审计日志（认证、授权）
- [ ] 敏感数据脱敏

### 监控指标

- [ ] 请求计数
- [ ] 响应时间
- [ ] 错误率
- [ ] 资源使用率
- [ ] 告警规则配置

---

## 🚀 部署检查

### 基础设施

- [ ] HTTPS 证书有效
- [ ] DNS 解析正确
- [ ] 防火墙规则配置
- [ ] 负载均衡健康检查
- [ ] 自动扩缩容配置

### 高可用

- [ ] 多实例部署
- [ ] 数据库主从复制
- [ ] 故障转移机制
- [ ] 备份恢复测试

## ✅ 功能测试

### 基本功能

- [ ] Agent Card 可获取
- [ ] message/send 正常工作
- [ ] 多轮对话 contextId 正确
- [ ] 文件上传/下载正常
- [ ] SSE 流式响应正常

### 异常处理

- [ ] 无效 JSON 返回错误
- [ ] 缺失字段返回错误
- [ ] 认证失败返回 401
- [ ] 权限不足返回 403
- [ ] 超时正确处理
